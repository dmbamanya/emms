MANUAL DEL USUARIO

4. Operaciones con Clientes

4.1 Buscar y ver datos de cliente

Esta función presenta un formulario para la búsqueda de clientes activos o inactivos. Todos los campos o filtros son opcionales. Los filtros para nombre y apellido aceptan llenado parcial. Para localizar un cliente y ver sus datos debe ir a:

MENU: Operaciones clientes > Buscar clientes

La búsqueda producirá una lista de resultados que muestra la cédula y el nombre completo de los clientes encontrados. La lista incluye además un par de hipervínculos mostrados en color resaltado para ver o editar los datos del cliente. Para ver los datos del cliente, hacer clic en el hipervínculo correspondiente de la lista de resultados.

Desde la barra de herramientas de la ficha de datos del cliente, haciendo clic convenientemente en los diferentes iconos, se pueden ejecutar las siguientes acciones:

  • Editar datos del cliente
  • Solicitar un préstamo
  • Ver listado de préstamos solicitados anteriormente
  • Cambiar de grupo, programa o asesor
  • Desactivar cliente - cliente sale del programa(1)
  • Activar cliente - cliente regresa al programa(1)
  • Registrar un negocio a nombre del cliente
  • Aplicar encuesta al cliente
  • Ver resultados de encuestas aplicadas
  • Ver historial de movimientos del cliente - cambios de grupo, salidas, etc.

(1) Solo aparecerá una de las opciones, en dependencia del estado del cliente en cuestión.

4.2 Agregar cliente nuevo

Esta función presenta un formulario que permite agregar un cliente nuevo al sistema. Los campos marcados con * son obligatorios. El sistema no admite cédulas (documento identidad nacional) duplicadas, si por error se digita una cédula ya registrada, el sistema devolverá el mensaje de error correspondiente. Para agregar un cliente nuevo debe ir a:

MENU: Operaciones clientes > Agregar cliente
Acerca del campo Grupo
Si el cliente no es miembro de algún grupo o banco, el campo Grupo se deja sin llenar. Si el cliente es miembro de un grupo o BDE, hacer clic en el icono de selección próximo al campo Grupo. Esto presentará una ventana flotante con una lista de grupos o BDEs para seleccionar mediante otro clic. La lista de grupos o BDEs disponibles en la lista de selección se limita a aquellos que se correspondan con los valores especificados en los campos Asesor y Programa. En otras palabras, antes de intentar la selección del BDE o grupo, es necesario haber especificado antes los campos Asesor y Programa. Para más información en relación a los grupos y/o BDEs vea el capítulo 5. Operaciones con Grupos
Tip
Una via alternativa para agregar un cliente nuevo a un Grupo, es a través de la ficha de datos del Grupo en cuestión. Comience por localizar la ficha correspondiente - vea el capítulo 5.1 Buscar y ver datos de un Grupo o BDE, luego desde la ficha de datos del grupo, haga clic en el icono Agregar cliente nuevo.

El sistema guarda registro de la fecha de creación y de la última edición de los datos de cada cliente, así como de los usuarios autores de la creación y la edición.

4.3 Editar datos de cliente

Siga las mismas instrucciones descritas en el capítulo 4.1 Buscar y ver datos de cliente, pero en el último paso, en la lista de resultados, hacer clic al hipervínculo Editar en lugar de Ver.

El formulario para editar los datos del cliente es el mismo que se usa para agregar clientes nuevos - vea capítulo 4.2 Agregar cliente nuevo, con la diferencia de que los campos Programa, Asesor y Grupo están bloqueados.

Para efectuar cambios en alguno de los campos bloqueados vea los dos próximos epígrages.

4.4 Mover cliente de grupo, asesor o programa

Los cambios de grupo, asesor o programa deben realizarse desde la ficha de datos del cliente. Vea las instrucciones descritas en el epígrafe 4.1 Buscar y ver datos de cliente. Una vez en la ficha del cliente buscar el icono Cambiar grupo de la barra de herramientas y seguir las instrucciones del formulario.

4.5 Desactivar Cliente

La desactivación debe realizarse desde la ficha de datos del cliente. Vea las instrucciones descritas en el epígrafe 4.1 Buscar y ver datos de cliente. Una vez en la ficha del cliente buscar el icono Desactivar de la barra de herramientas y seguir las instrucciones del formulario. Este preparado para especificar y describir el motivo de la desactivación.

Además del procedimiento anterior, un cliente también puede quedar desactivado si se desactiva el grupo o BDE al que pertenece. Para más información vea el epígrafe correspondiente a la desactivación de grupos y/o BDEs.

4.6 Activar Cliente

La Activación debe realizarse desde la ficha de datos del cliente. Vea las instrucciones descritas en el epígrafe 4.1 Buscar y ver datos de cliente. Una vez en la ficha del cliente buscar el icono Activar de la barra de herramientas y seguir las instrucciones del formulario.

4.7 Otras operaciones con clientes

Otras operaciones con clientes son:

  • Solicitar un préstamo
  • Ver listado de préstamos solicitados anteriormente
  • Registrar un negocio a nombre del cliente
  • Aplicar encuesta al cliente
  • Ver resultados de encuestas aplicadas
  • Ver historial de movimientos del cliente - cambios de grupo, salidas, etc.

Todas estas operaciones pueden ser llamadas desde la ficha de datos del cliente.
