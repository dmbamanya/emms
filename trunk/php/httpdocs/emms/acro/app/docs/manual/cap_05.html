MANUAL DEL USUARIO

5. Operaciones con Grupos

El e-MMS incluye soporte para registrar diferentes tipos de asociaciones entre los clientes. Los tipos de asociaciones posibles son:

Grupos
Los préstamos a los miembros de un Grupo tendrán responsabilidad y seguimiento individual, o sea, el sistema lleva registro de los préstamos a nivel personal, aunque permite la gestión de diversas tareas a nivel de Grupo.
Centro o BDE, en lo adelante BDE.
Estas son asociaciones entre entre Grupos. Los clientes miembros de un Grupo, son indirectamente miembros del BDE al que pertenece el Grupo, y son elegibles para accesar a los fondos del BDE y demás facilidades y responsabilidades dentro de un BDE.

5.1 Buscar y ver datos de un Grupo o BDE

Para localizar un grupo o BDE y ver sus datos debe ir a:

MENU: Operaciones grupos > Buscar grupo/centro

Este formulario permite la búsqueda de Grupos o BDEs. Todos los campos o filtros son opcionales, la búsqueda producirá una lista de resultados que muestra la código y el nombre de los Grupos o BDEs encontrados. La lista incluye además un par de hipervínculos mostrados en color resaltado para ver o editar los datos del Grupo o BDE. Para ver los datos del Grupo o BDE, hacer clic en el hipervínculo correspondiente de la lista de resultados. Para más información vea los epígrafes siguientes.

Ficha de datos de BDE

Desde la barra de herramientas de la ficha de datos del BDE, haciendo clic convenientemente en los diferentes iconos, se pueden ejecutar las siguientes acciones:

  • Editar datos del BDE
  • Agregar un nuevo Grupo como miembro del BDE
  • Desactivar BDE. El BDE completo, incluyendo todos los grupos y asociados correspondientes, salen del programa

Ficha de datos de Grupo

Desde la barra de herramientas de la ficha de datos del Grupo, haciendo clic convenientemente en los diferentes iconos, se pueden ejecutar las siguientes acciones:

  • Editar datos del Grupo
  • Agregar un nuevo miembro - asociado
  • Mover Grupo hacia otro BDE - hereda asesor y programa del nuevo BDE
  • Desactivar grupo - Grupo completo, incluyendo todos los miembros, salen del programa

5.2 Agregar un nuevo Grupo o BDE

Para agregar un nuevo Grupo o BDE debe ir a:

Operaciones grupos/BDEs > Agregar grupo/BDE

Como paso preliminar el sistema le exige especificar el tipo de grupo a crear - Grupo o BDE. El sistema presentará entonces el formulario correspondiente según el tipo seleccionado.

Nota:
La manera más rápida para crear un nuevo BDE con sus correspondientes Grupos, es proceder con la creación del BDE primero, esto le permitirá crear automáticamente hasta 5 Grupos miembros del BDE en cuestión, sin necesidad de operaciones adicionales. En cualquier caso, los clientes miembros de cada Grupo deberán ser incluidos después. Para más información vea los epígrafes siguientes.

Formulario para creación de BDE

Este formulario permite la entrada de datos para agregar un BDE nuevo al sistema. Los campos marcados con * son obligatorios. El sistema no permite crear un BDE sin Grupos miembros, en el formulario de creación del BDE deberá especificar cuántos (hasta un máx. de 5) Grupos conformarán inicialmente al BDE. Estos Grupos serán nombrados automáticamente por el sistema siguiendo la secuencia Grupo 1, Grupo 2, Grupo 3, etc. El usuario podrá editar las fichas de los Grupos creados automáticamente para asignarles nombres descriptivos y agregar clientes miembros.

Formulario para creación de Grupo

Este formulario permite la entrada de datos para agregar un Grupo nuevo al sistema. Los campos marcados con * son obligatorios. El sistema no permite crear un Grupo independiente, en otras palabras, esté preparado para asociar el Grupo a un BDE ya existente.

5.3 Editar datos de Grupo o BDE existente

Para localizar un Grupo o BDE existente y editar sus datos debe ir a:

Operaciones grupos: Buscar grupo/centro

Siga las mismas instrucciones descritas anteriormente en el capítulo 5.1 Buscar y ver datos de Grupo o BDE, pero en el último paso, en la lista de resultados, hacer clic al hipervínculo Editar en lugar de Ver. Con esto se presentará el formulario correspondiente donde podrá realizar los cambios necesarios. Para más información vea los capítulos siguientes.

Formulario para editar datos de BDE existente

Acerca de los campos Asesor y Programa
La modificación de estos campos afectará a todos los Grupos y clientes registrados bajo el BDE, los cuales heredarán como propios el nuevo Asesor y/o Programa seleccionado.
Acerca de los campos Presidente, Tesorero y Secretario
Estos campos permiten especificar la Junta del BDE. Solo aparecen disponibles si hay clientes registrados bajo el BDE en cuestión.

Formulario para editar Grupo existente

Acerca de los campos Asesor y Programa
La modificación directa de estos campos no está permitida para los Grupos, sin embargo puede mover o trasladar el Grupo hacia otro BDE registrado con el Asesor y Programa requerido. Para más información vea el epígrafe dedicado a los traslados de Grupos.
Acerca de los campos Presidente, Tesorero y Secretario
Estos campos permiten especificar la Junta del Grupo. Solo aparecen disponibles si hay clientes registrados bajo el Grupo en cuestión.

5.4 Otras operaciones con Grupos y/o BDEs

Otras operaciones con Grupos y/o BDEs son:

  • Mover Grupo hacia otro BDE. El Grupo y todos los clientes miembros heredan asesor y programa del nuevo BDE.
  • Desactivar Grupo o BDE. Todos los miembros salen del programa, no se admite deshacer esta operación. Este preparado para especificar causa de la salida. Para re-ingresar los clientes al programa tendrá que seguir el procedimiento indicado en el capítulo 4.6 Activar Cliente.

Estas operaciones se realizan desde la ficha de datos del Grupo o BDE
